\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class sf\+::\+Vector2$<$ T $>$}
Utility template class for manipulating 2-\/dimensional vectors. 

\doxylink{a01953}{sf\+::\+Vector2} is a simple class that defines a mathematical vector with two coordinates (x and y). It can be used to represent anything that has two dimensions\+: a size, a point, a velocity, etc.

The template parameter T is the type of the coordinates. It can be any type that supports arithmetic operations (+, -\/, /, \texorpdfstring{$\ast$}{*}) and comparisons (==, !=), for example int or float.

You generally don\textquotesingle{}t have to care about the templated form (sf\+::\+Vector2$<$\+T$>$), the most common specializations have special typedefs\+: \begin{DoxyItemize}
\item sf\+::\+Vector2$<$float$>$ is \doxylink{a00669_acf03098c2577b869e2fa6836cc48f1a0}{sf\+::\+Vector2f} \item sf\+::\+Vector2$<$int$>$ is \doxylink{a00669_ace09dd1447d74c6e9ba56ae874c094e1}{sf\+::\+Vector2i} \item sf\+::\+Vector2$<$unsigned int$>$ is \doxylink{a00669_aaa02ba42bf79b001a376fe9d79254cb3}{sf\+::\+Vector2u}\end{DoxyItemize}
The \doxylink{a01953}{sf\+::\+Vector2} class has a small and simple interface, its x and y members can be accessed directly (there are no accessors like set\+X(), get\+X()) and it contains no mathematical function like dot product, cross product, length, etc.

Usage example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{a01953}{sf::Vector2f}}\ v1(16.5f,\ 24.f);}
\DoxyCodeLine{v1.x\ =\ 18.2f;}
\DoxyCodeLine{\textcolor{keywordtype}{float}\ \mbox{\hyperlink{a01953_a420f2481b015f4eb929c75f2af564299}{y}}\ =\ v1.y;}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{a01953}{sf::Vector2f}}\ v2\ =\ v1\ *\ 5.f;}
\DoxyCodeLine{\mbox{\hyperlink{a01953}{sf::Vector2f}}\ v3;}
\DoxyCodeLine{v3\ =\ v1\ +\ v2;}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{bool}\ different\ =\ (v2\ !=\ v3);}

\end{DoxyCode}


Note\+: for 3-\/dimensional vectors, see \doxylink{a01957}{sf\+::\+Vector3}.
