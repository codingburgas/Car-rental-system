SFML is a simple, fast, cross-\/platform and object-\/oriented multimedia API. It provides access to windowing, graphics, audio and network. It is written in C++, and has bindings for various languages such as C, .Net, Ruby, Python.\hypertarget{a02196_autotoc_md5}{}\doxysubsection{\texorpdfstring{Authors}{Authors}}\label{a02196_autotoc_md5}
