\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class sf\+::\+Vector3$<$ T $>$}
Utility template class for manipulating 3-\/dimensional vectors. 

\doxylink{a01957}{sf\+::\+Vector3} is a simple class that defines a mathematical vector with three coordinates (x, y and z). It can be used to represent anything that has three dimensions\+: a size, a point, a velocity, etc.

The template parameter T is the type of the coordinates. It can be any type that supports arithmetic operations (+, -\/, /, \texorpdfstring{$\ast$}{*}) and comparisons (==, !=), for example int or float.

You generally don\textquotesingle{}t have to care about the templated form (sf\+::\+Vector3$<$\+T$>$), the most common specializations have special typedefs\+: \begin{DoxyItemize}
\item sf\+::\+Vector3$<$float$>$ is \doxylink{a00669_af97357d7d32e7d6a700d03be2f3b4811}{sf\+::\+Vector3f} \item sf\+::\+Vector3$<$int$>$ is \doxylink{a00669_ad066a8774efaf7b623df8909ba219dc7}{sf\+::\+Vector3i}\end{DoxyItemize}
The \doxylink{a01957}{sf\+::\+Vector3} class has a small and simple interface, its x and y members can be accessed directly (there are no accessors like set\+X(), get\+X()) and it contains no mathematical function like dot product, cross product, length, etc.

Usage example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{a01957}{sf::Vector3f}}\ v1(16.5f,\ 24.f,\ -\/8.2f);}
\DoxyCodeLine{v1.x\ =\ 18.2f;}
\DoxyCodeLine{\textcolor{keywordtype}{float}\ \mbox{\hyperlink{a01957_a6590d50ccb862c5efc5512e974e9b794}{y}}\ =\ v1.y;}
\DoxyCodeLine{\textcolor{keywordtype}{float}\ \mbox{\hyperlink{a01957_a2f36ab4b552c028e3a9734c1ad4df7d1}{z}}\ =\ v1.z;}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{a01957}{sf::Vector3f}}\ v2\ =\ v1\ *\ 5.f;}
\DoxyCodeLine{\mbox{\hyperlink{a01957}{sf::Vector3f}}\ v3;}
\DoxyCodeLine{v3\ =\ v1\ +\ v2;}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{bool}\ different\ =\ (v2\ !=\ v3);}

\end{DoxyCode}


Note\+: for 2-\/dimensional vectors, see \doxylink{a01953}{sf\+::\+Vector2}.
